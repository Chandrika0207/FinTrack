# FinTrack – Finance Tracking System
Project Description

FinTrack is a web-based finance tracking system developed using HTML, CSS, PHP, and MySQL. It helps users manage their financial records by tracking income, expenses, and transaction history in a structured and efficient manner. The system provides an intuitive interface for recording transactions and maintaining financial data securely through a database.

Technologies Used
HTML
CSS
PHP
MySQL
Features
Add and manage income records.
Track and categorize expenses.
View transaction history.
Update and delete financial records.
Store data securely using MySQL.
User-friendly web interface.
Objectives
Simplify financial record management.
Reduce manual bookkeeping efforts.
Maintain accurate income and expense records.
Provide quick access to financial data.
Improve financial monitoring and analysis.
